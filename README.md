# cortextos-framework

korendigital's fork of [grandamenium/cortextos](https://github.com/grandamenium/cortextos).

## Workflow

- All changes via Pull Request — never push directly to `main`
- Upstream changes flow in via the `upstream-sync` GitHub Action (PR-only)
- CI runs lint/types/tests on every PR
- Claude AI security review runs on every PR

See [ops-architecture.md](orgs/korendigital/vault/02-areas/SOPs/operations/ops-architecture.md) for the full 4-rule operating doctrine.
